# Icon Size Requirements

[https://electronjs.org/docs/api/native-image#supported-formats]

**Small icon**
16x16 (100% DPI scale)
20x20 (125% DPI scale)
24x24 (150% DPI scale)
32x32 (200% DPI scale)

**Large icon**
32x32 (100% DPI scale)
40x40 (125% DPI scale)
48x48 (150% DPI scale)
64x64 (200% DPI scale)
256x256
